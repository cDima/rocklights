# Rocklights app by Dmitry Sadakov

A node.js app for synced lights using the [einaros/ws](http://einaros.github.io/ws/) WebSockets implementation.

## Running Locally

``` bash
npm install
npm start
```

## Running on Heroku

``` bash
heroku create
git push heroku master
heroku open
```
